# Desafio 2

O grande mercado anual do Reino Nova Era está se aproximando e os comerciantes precisam atualizar o valor de seus produtos. O Rei decretou que todos os itens da categoria 'Armaduras' que atualmente custam mais de 500 moedas de ouro devem receber um subsídio real, aplicando-se um desconto de 20% no preço original para incentivar a defesa dos cidadãos.
<br><br>
**Sua Tarefa:** Mostre o nome do item e o novo preço com o desconto aplicado (com a precisão de duas casas decimais) apenas para os itens que se enquadram nessa regra. Ordene a lista de forma decrescente, mostrando primeiro o item que ficou mais caro após o desconto.
